# 🤖 サブエージェント作業計画

**作成日：** 2026-02-06
**目的：** 並列作業による効率化

## 🎯 エージェント役割分担

### Agent-A: TALES詳細分析エージェント
**タスク：**
- TALESサイトの入賞作品詳細読み込み
- 各ジャンル別の特徴抽出
- 成功パターンの定量的分析
- コンテスト傾向の時系列分析

**成果物：**
- 入賞作品分析レポート
- ジャンル別成功法則まとめ
- コンテスト攻略ガイド

### Agent-B: 人気小説研究エージェント
**タスク：**
- 世の中で人気の小説調査（近年のベストセラー等）
- 映像化された小説の分析
- トレンドワードとテーマの抽出
- 読者嗜好の変化調査

**成果物：**
- 人気小説分析レポート
- 映像化適性評価リスト
- トレンドキーワード集

### Agent-C: 創作支援エージェント
**タスク：**
- プロット生成システムの開発
- キャラクター設定テンプレート作成
- 文章校正・推敲支援ツール準備
- 創作効率化ワークフロー設計

**成果物：**
- 創作支援ツール群
- プロットテンプレート
- 執筆ガイドライン

## ⏰ 作業スケジュール

### Day 1-2: 分析フェーズ
- Agent-A: TALES分析開始
- Agent-B: 人気小説リサーチ開始
- メインエージェント: 全体統括・方向性調整

### Day 3-4: 統合・企画フェーズ
- 各エージェントの成果統合
- コンセプト・テーマ決定
- Agent-C: 創作ツール準備

### Day 5-7: 創作フェーズ
- プロット作成
- 執筆開始
- 並行してAgent-C: 映像化準備

## 📊 成功指標

- 各エージェントのタスク完了率
- 分析データの質と量
- 創作物の完成度
- 最終的なTALES投稿までの期間

## 🔄 連携方法

- 共有ファイル経由でのデータ交換
- 定期的な進捗同期（6時間毎）
- Git commitでの進捗記録
- 問題発生時の即座のエスカレーション
